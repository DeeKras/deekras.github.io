##Dictionaries

###ways to create the dictionary

```python

>>> d = dict(zip(('A','B','C'),(1,2,3)))
>>> d
{'A': 1, 'C': 3, 'B': 2}
```

```python

>>> e = {}
>>> e['A']= 1
>>> e
{'A': 1}
>>> e['B']=2
>>> e
{'A': 1, 'B': 2}
>>> e['C']=3
>>> e
{'A': 1, 'C': 3, 'B': 2}
```

```python

>>> e= {}
>>> zipped = zip(('A','B','C'),(1,2,3))
>>> for key, value in zipped:
...   e[key] = value
... 
>>> e
{'A': 1, 'C': 3, 'B': 2}
```
